#  FitCore - Sistema Integrado de Gestão de Academia

## Objetivo do Projeto

O **FitCore** foi desenvolvido como parte da disciplina de **Projeto e Desenvolvimento de Software**, com o objetivo de aplicar conceitos de engenharia de software na construção de um sistema real.

O sistema integra funcionalidades essenciais para academias, incluindo:

- Gestão de alunos  
- Controle financeiro  
- Planejamento de treinos  
- Monitoramento de frequência  

A proposta é transformar processos manuais em um sistema estruturado, escalável e orientado a dados.

---

##  Fundamentação Teórica

O projeto se baseia nos princípios de:

###  Engenharia de Software

- Modelagem de sistemas para redução de complexidade  
- Organização de processos e dados  
- Evolução e manutenção contínua  

### Orientação a Objetos

- Classes representam entidades do mundo real  
- Objetos interagem por meio de operações  
- Uso de abstração, encapsulamento e herança  

Um sistema de software é uma combinação de dados, processos e pessoas para gerar valor organizacional.

---

##  Modelagem do Sistema (UML)

O sistema foi modelado utilizando UML, com foco na representação estrutural e comportamental.

###  Diagrama de Classes

Principais entidades do sistema:

- Aluno  
- Treino  
- Exercício  
- Plano  
- Pagamento  
- Professor  

###  Relacionamentos

- Um aluno pode possuir vários treinos  
- Um plano pode estar associado a vários alunos  
- Um treino é composto por vários exercícios  
- Pagamentos estão vinculados a alunos  

---

##  Estrutura do Sistema

### 1. Entrada de Dados

O sistema recebe:

- Dados cadastrais de alunos  
- Informações de planos  
- Definição de treinos  
- Registros de pagamento  

---

### 2. Processamento

O sistema realiza:

- Associação de alunos a planos  
- Vinculação de treinos personalizados  
- Cálculo de status financeiro  
- Registro e controle de frequência  

---

### 3. Saída

O sistema gera:

- Relatórios financeiros  
- Histórico de treinos  
- Situação do aluno (ativo/inadimplente)  
- Indicadores de uso da academia  

---

##  Funcionalidades

- Cadastro e gerenciamento de alunos

- Controle de pagamentos  
- Monitoramento de frequência
